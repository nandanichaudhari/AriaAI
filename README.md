## 🎯 About AriaAI

**AriaAI** is an immersive, futuristic web portal powered by **Omnidimension AI technology**. This intelligent assistant provides real-time conversations, smart insights, and a stunning glass-morphism interface with neon glow effects.

### ✨ Key Features

- 🧠 **Intelligent AI Agent** - Powered by Omnidimension's advanced language model
- 🏠 **Smart Return Navigation** - After chatting, easily return to home screen
- 💎 **Glass Morphism Design** - Modern, futuristic UI with blur effects
- ⚡ **Real-time Responses** - Instant AI-powered conversations
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile
- 🎨 **Animated Particles** - Dynamic background with floating particles
- 🔥 **Hover Effects & Glows** - Interactive elements with smooth animations

## 🚀 How to Use

1. **Chat with Aria** - Scroll down and click on the AI widget
2. **Ask Anything** - Get intelligent responses from the AI assistant
3. **Return to Home** - After conversation, click the prominent orange "Return to Home" button
